The reply for the above program is highly deterministic in nature and it does not answer on the basis of context of the conversation. It has limited knowledge and ask questions and provides replies based on the keywords present in user input. For example, it asks specific set of questions if it finds the keyword alcohol, irrespective of context.

I work in shop which sells scotch

Do you drink a lot of scotch?

The above reply is on account of the following actions by the program taken on user input.

  • Taking into account the sentiment assosiated with the user input
  • Incorporate models where it can learn replies from large corpus of human conversations and adapt to a different kinds of input
  • Trying to understand the context of the question rather than doing analysis on basis of certain keywords
